Instruções para a realização do Desafio

    Neste desafio você deverá criar um website que apresente quem é você e seus interesses. Você vai criar uma espécie de página de Linkedin ou de CV personalizada.

  • Minha História
  • Meus Interesses
  • Minha experiência

    O website deve conter:

  • Uma barra de navegação onde possamos acessar cada uma das seções acima
  • Títulos e conteúdo de cada seção
  • Rodapé
  • Uma tag de analytics instalada
